# TaskNet

**TaskNet** is a dynamic web-based system built with **Java technologies** such as **JSP**, **Servlets**, and **JDBC**, connected to a **MySQL database**. Users can register, log in, and manage a personal todo list by performing Create, Read, Update, and Delete (CRUD) operations on tasks with simple and user-friendly interface. 

This project is ideal for learning **Java web development**, **Servlets/JSP**, and **basic project management** features.


## Features

- User-friendly interface to manage tasks
- Add, edit, and delete tasks
- Task status tracking (completed, pending)
- Responsive layout (works in modern browsers)
- Built using standard **Java Servlets, JSP, and HTML/CSS**
- Works with **Apache Tomcat 9** server


## Technologies Used

- **Backend:** Java, Servlets, JSP
- **Frontend:** HTML, CSS, JavaScript
- **Connectivity:** JDBC
- **Database:** MySQL  
- **IDE:** Apache NetBeans
- **Server:** Apache Tomcat 9
- **Version Control:** Git/GitHub

## Usage

- Open the Todo App in the browser
- Create a new task by entering a title and description
- Edit tasks to update details
- Delete tasks you no longer need
- Track task completion status

## Software Testing

TaskNet was tested to ensure correctness and reliability using **JUnit** and **Selenium WebDriver**.

- **JUnit Testing:**  
  - Boundary Value Analysis for username and password  
  - Equivalence Class Testing for valid and invalid inputs  
  - Robustness Testing for null or empty fields  

- **Selenium Testing:**  
  - Automated login tests (valid/invalid inputs)  
  - Task creation, update, and deletion  
  - UI navigation and logout functionality  

**Tools:** JUnit, Selenium WebDriver, ChromeDriver, Apache Tomcat 9
